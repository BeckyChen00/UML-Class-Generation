---------------------4/20------Board Game:
------------------------------------------------------------
AI:
Enumerations:
Color(Black, Yellow)
GameState(Block Selection, Block Replacement, Block Movement)

Classes:
Board(int width, int height, Color color)
Block(int side, int width, int height, Color color)
SquareBlock(int side, Color color)
RectangularBlock(int width, int height, Color color)

Relationships:
1 Board contain *Block
1 Block associate 1 Color
1 Block associate 1 GameState
1 Block associate *Block (for no overlapping)
1 Block associate *Block (for minimum distance)
Block inherit SquareBlock
Block inherit RectangularBlock

Actions:
Block Selection
Input
Left-clicked down on a block.
Processing
The game state becomes "Block replacement."
Output None.

Block Deselection
Input
Left-clicked up on a selected block.
Processing
The game state becomes "Block selection."
Output None.

Block Movement
Input
Mouse movement during the "Block movement" state.
Processing
The selected block follows the mouse movement without overlapping the other blocks and exits the game zone. The selected block can't move near other blocks at least 0.0x from the other blocks.
Output None.

Undo Action
Input Menu selection.
Processing
The game displays the block positions at the places where they were before the last movement. This action is considered a movement. The "undo" action is unavailable if there is no previous movement.
Output New game board display.
